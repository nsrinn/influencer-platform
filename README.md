## ✅ Features
- Role-based auth (Admin + Influencer) with JWT
- Unique affiliate link per influencer with click tracking
- Automatic 10% commission calculation on every sale
- Payment management — Pending → Approved → Paid
- Line chart — Revenue over time
- Bar chart — Top influencers by revenue
- Pie chart — Revenue split across influencers
- AI Feature 1 — Performance insights per influencer
- AI Feature 2 — Sales prediction (7-day forecast)
- AI Feature 3 — Fraud detection with risk scoring
- Fully deployed on Vercel + Railway + Neon PostgreSQL

## 🛠 Tech Stack
- Frontend: React, Recharts, React Router
- Backend: Node.js, Express, Sequelize
- Database: PostgreSQL (Neon)
- AI: Claude API (Anthropic)
- Deploy: Vercel (frontend) + Railway (backend)
